AI-News-Briefing sends news reports to your Notion workspace. You need to connect it to Notion using a token.

4. Share the Notion page or database where you want news to appear with the integration you created.

### How to add your token to AI-News-Briefing:

1. Open the AI-News-Briefing app.

2. Look for the settings or configuration option.

3. Paste your Notion token where it asks for API access.

4. Enter the Notion database ID or link where news should be saved.

5. Save your settings.
